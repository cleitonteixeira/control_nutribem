# Gestek 🏭

**Gestek** é um sistema web de gerenciamento de equipamentos desenvolvido com Django.

## 🎯 Sobre o Projeto

Gestek é uma solução corporativa para centralizar e gerenciar o inventário de equipamentos de sua organização. O sistema permite controle completo do ciclo de vida dos equipamentos, desde registro até manutenção e descarte.

## ✨ Funcionalidades Principais

- ✅ Cadastro e gerenciamento de equipamentos
- ✅ Rastreamento de localização e status
- ✅ Registro de manutenções e reparos
- ✅ Importação/exportação de dados em Excel
- ✅ Relatórios e análises
- ✅ Histórico de movimentações
- ✅ Controle de usuários e permissões

## 🛠️ Stack Tecnológico

- **Backend:** Django 5.2.13
- **Linguagem:** Python 3.x
- **Processamento de dados:** openpyxl (Excel)
- **Banco de dados:** SQLite/PostgreSQL

## 📋 Pré-requisitos

- Python 3.8+
- pip
- virtualenv (recomendado)
